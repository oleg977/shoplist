# ShopList

ShopList — это веб-сайт на Django, созданный для публикации информации о физических товарах и адресах магазинов, где их можно приобрести. Проект разработан для локальных IT-интеграторов, помогая розничным сетям представлять свой ассортимент без необходимости вести полноценный интернет-магазин.

## Функциональные возможности

*   **Просмотр товаров:** Пользователи могут просматривать список товаров с названием, ценой, описанием и изображением.
*   **Детали товара:** Возможность посмотреть расширенную информацию о товаре, включая список адресов магазинов, где он доступен.
*   **Поиск товаров:** Поиск по названию товара.
*   **Аутентификация и роли:**
    *   **Пользователь (user):** Может просматривать товары.
    *   **Менеджер (sales executive):** Может просматривать, добавлять, редактировать и удалять товары.
    *   **Администратор (admin):** Имеет полный доступ к системе (через Django Admin).
*   **Управление товарами:** Интерфейс для менеджеров и администраторов для добавления, редактирования и удаления товаров через веб-интерфейс.

## Нефункциональные требования

*   **Язык:** Python 3.10+
*   **Фреймворк:** Django 4.x
*   **Шаблоны:** Django Templates + Bootstrap
*   **База данных:** SQLite
*   **Адаптивный дизайн:** Совместимость с десктопными и мобильными устройствами.

## Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ваш-аккаунт/shoplist.git
    cd shoplist
    ```

2.  **Создайте и активируйте виртуальное окружение (рекомендуется):**
    ```bash
    python -m venv venv
    # На Windows:
    venv\Scripts\activate
    # На Linux/macOS:
    source venv/bin/activate
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
    (Предполагается, что вы создали `requirements.txt` командой `pip freeze > requirements.txt`)

4.  **Выполните миграции базы данных:**
    ```bash
    python manage.py migrate
    ```

5.  **Создайте суперпользователя (для доступа к Django Admin):**
    ```bash
    python manage.py createsuperuser
    ```

6.  **Запустите сервер разработки:**
    ```bash
    python manage.py runserver
    ```

7.  **Откройте браузер и перейдите по адресу:**
    ```
    http://127.0.0.1:8000/
    ```

## Использование

*   Для просмотра товаров не требуется вход в систему.
*   Для регистрации перейдите на `/users/register/`.
*   Для входа перейдите на `/users/login/`.
*   Менеджеры и администраторы могут управлять товарами через интерфейс на `/products/` (кнопки появляются после входа).
*   Администраторы могут управлять всем через Django Admin на `/admin/`.

## Структура проекта

*   `shoplist/`: Основной проект Django.
*   `users/`: Приложение для управления пользователями, аутентификацией и ролями.
*   `products/`: Приложение для управления товарами, изображениями и адресами магазинов.
*   `media/`: Папка для загружаемых файлов (изображений товаров). *Важно: при деплое настраивается отдельно.*
*   `static/`: Папка для статических файлов (CSS, JS, изображения проекта). *Можно добавить при необходимости.*
